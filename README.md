## Automated Segmentation:

To run the code:

+ clone repo, using `git clone https://github.com/iAkashP/Bacteria_strip_segmentation`.
+ Copy the required Test images into `/Test_Images/` directory.
+ python `Bact_Strip_Segmentation.py`.
+ check `/Results/`.




**For more info** go through [Bacteria_strip_readme.pdf](https://github.com/iAkashP/Bacteria_strip_segmentation/blob/master/Bacteria_strip_readme.pdf).
